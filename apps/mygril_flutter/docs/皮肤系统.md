# 皮肤系统（小白版）

这份文档用最简单的语言，解释什么是"皮肤系统"以及它能帮你做什么。

---

## 🤔 什么是皮肤系统？

**打个比方**：就像手机可以换壁纸和主题一样，皮肤系统让整个 App 可以"换装"。

换一套皮肤，这些东西都会变：
- 颜色（主色调、背景色、按钮颜色）
- 形状（圆角大不大、边框粗不粗）
- 特效（要不要毛玻璃模糊效果）

---

## 🎨 目前有哪些皮肤？

### 1. MoeTalk 风格（默认，已实现）

**长什么样**：
- 粉色顶部栏
- 蓝色主按钮
- 圆角比较小（10px）
- 边框很细（0.5px）
- 简洁清爽，没有特效

**适合**：喜欢简约风格的用户

---

### 2. 毛玻璃现代风格（计划中）

**长什么样**：
- 半透明背景
- 背后有模糊效果（像雾面玻璃）
- 圆角比较大（24px）
- 渐变紫蓝色调

**适合**：喜欢科技感、现代感的用户

---

### 3. 森林之书风格（计划中）

**长什么样**：
- 森林绿色调
- 像旧书页一样的纸张质感
- 圆角比较小
- 边框稍微粗一点

**适合**：喜欢自然、复古风格的用户

| 界面部分 | 会变什么 |
|---------|---------|
| **顶部栏** | 背景色、边框 |
| **提示气泡** | 背景色、圆角 |
| **聊天气泡** | 背景色、圆角 |
| **底部导航** | 背景色、边框 |
| **输入框** | 边框、圆角 |
| **联系人列表** | 分割线粗细 |
| **设置面板** | 背景色、边框 |

## 🛠️ 技术说明（给未来的我看）

**相关文件位置**：

| 文件 | 作用 |
|------|------|
| `lib/src/core/theme/skin_config.dart` | 皮肤配置的"模板"，定义皮肤需要有哪些属性 |
| `lib/src/core/theme/skin_provider.dart` | 管理当前选中的皮肤 |
| `lib/src/core/theme/skins/moetalk_skin.dart` | MoeTalk 风格的具体配置 |
| `lib/src/core/theme/tokens.dart` | 颜色常量定义（MoeColors） |

**皮肤系统的工作原理**：

1. **皮肤配置**（SkinConfig）：定义一套皮肤长什么样
   - 用什么颜色
   - 圆角多大
   - 边框多粗
   - 要不要模糊效果

2. **皮肤管理器**（SkinProvider）：记住当前选的是哪套皮肤

3. **界面组件**：从皮肤配置里读取样式，自动应用

**添加新皮肤只需要**：
1. 在 `skins/` 文件夹下写一个新文件（比如 `forest_book_skin.dart`）
2. 在 `skin_provider.dart` 的皮肤列表里注册它
3. 完成！所有界面自动生效
